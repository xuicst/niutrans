## 概述
这是ONLYOFFICE编辑器插件源项目，轻松将bilibili视频嵌入到您的文档中。
该插件使用[bilibili_open_api](https://open.bilibili.com/doc)允许嵌入bilibili视频播放器并管理JavaScript的使用。
> 默认情况下，它安装在云、自托管和桌面版本的ONLYOFFICE编辑器中。
## 如何使用
* 在“插件”选项卡中找到BiliBili插件。
* 插入视频URL，然后按OK。
* 图像形式的视频预览将出现在您的文档中，单击它以播放视频。
如果您需要有关如何使用或编写自己的插件的更多信息，请参阅此https://api.onlyoffice.com/plugin/basic
